<h2 id="descricao-do-projeto">Descrição do Projeto</h2>
<p>Desafio proposto pela Alura em parceria com a Oracle, com intuito de testar nossos conhecimentos em Java, Orientação Objeto , Spring e consumo de APIs.
<p>O projeto com intuito de consumir a API Gutendex , e a partir disso consumir ela , e preencher o banco de dados postgree, e a partir disso gerar buscas de dados armazenados nele</p>
